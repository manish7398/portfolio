# Manish Pal - Portfolio

A modern, responsive portfolio website for a MERN Stack Developer.

## Deployment to GitHub Pages

1. **Create a GitHub Repository**
   - Go to [GitHub](https://github.com) and create a new repository named `manish-pal-portfolio`

2. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git branch -M main
   git remote add origin https://github.com/yourusername/manish-pal-portfolio.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click **Settings** → **Pages** (in left sidebar)
   - Under **Source**, select **Deploy from a branch**
   - Under **Branch**, select `main` and folder `/ (root)`
   - Click **Save**
   - Wait 1-2 minutes, your site will be live at: `https://yourusername.github.io/manish-pal-portfolio/`

## Customization

- Edit `index.html` to update any information
- To add your photo: replace the profile image section with your own photo
- To add project demo URLs: find the project-links section in index.html
